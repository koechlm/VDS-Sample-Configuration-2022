V.6.3 Create/Edit Persons
Vault QuickStart uses the custom object "Persons" to link with other objects like Organisations or Tasks. Moreover, a person can also be linked to a document.
Create   
Persons are numbered sequentially, as long as the object numbering of this type is active. The first and last name are required fields.
 
If the "Indent Number" property is assigned to the Custom Object category the generated number will be stored in this property. For the naming of the Custom Object the combination of first and last name is used.
 
If the numbering scheme is disabled by the administrator you can assign manually a number or text for the Persons naming. The first and last name are required fields.
Note: For the naming of the Custom Object the combination of first and last name is used and will override the entered Name value.
