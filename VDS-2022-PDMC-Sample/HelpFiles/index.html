Autodesk Vault QuickStart is a pre-configured environment for Vault Workgroup, Professional or Office. This configuration is based on preferred working methods and procedures used by SMEs in mechanical engineering.
This documentation describes the user interface customizations and user experience. Note that these customizations can be modified or replaced by Vault administrators.
