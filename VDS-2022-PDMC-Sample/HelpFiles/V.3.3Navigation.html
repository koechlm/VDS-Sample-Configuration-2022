V.3.3 Navigation
You can use the usual keys to navigate within and close the dialog box:
1 Tab - move from field to field
2 Enter for line breaks within the field description (multi line text)
3 F1 to access the help files
4 Enter key close and save the changes to the file
5 Esc to cancel and close the dialog box
