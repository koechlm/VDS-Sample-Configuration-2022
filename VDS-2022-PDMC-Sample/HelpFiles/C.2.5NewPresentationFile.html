C.2.5 New Presentation File
 
The first time you save presentations an Inventor dialog box with corresponding property fields will appear.
The dialog box (drawings and presentations) will respect the option “Generate file numbers for drawing and presentation files”:
  
 
Option - enabled: 
1 - 7 The functionality of the dialog is the same as the new components dialog.
 
Note: Properties: These are - if available - taken from the model, but can be overridden for the presentation file.
 
 
Option - disabled:
Location and file name are taken from the 3D model. Consequently, there are no shortcuts to the selection of project-storage locations offered.
 
1 The folder is set to the model file's folder
 
(2) There are no shortcuts to the selection of project-storage locations offered
 
3 The Category is pre-selected:
  • For the Vault Quickstart configuration the category is set to "3D Component".
  • For Other Vault Configurations the category is set to "Engineering".
 
4 As the filename is taken from the model, these options are disabled
 
5 The category properties are organized in a tabbed grid. This grid can be easily extended with extra tabs: ERP/ITEM tab and Terms catalog tab.
More information in the extensions section.
 
Note: Properties: These are - if available - taken from the model, but can be overridden for the presentation file.
 
6 The filename is pre-set to model’s file name. Editing of the filename is allowed.
 
7 Access the Help Files
 
Special case: Saving an IPN file without a linked model
 
In case the option "Generate file numbers for drawing and presentation files" is enabled the standard save new presentation file dialog will show. (see above)
Note: Properties: in this case, no properties can be read from the model and should be entered by the user.
