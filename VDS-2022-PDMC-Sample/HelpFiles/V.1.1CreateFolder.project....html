V.1.1 Create Folder / project...
Select project or folder via the category drop-down.
1 When creating a standard Folder the Name input field is mandatory. Projects are numbered sequentially. The numbering applies throughout the vault, regardless of the directory structure. The project number is also the name of the project directory.
 
Note: When switching back from project to folder the Name input field will be shown again in the Quickstart VDS configuration. The Default Vault Data Standard dialog does not allow to switch back from a category using numbering scheme to text input field.
 
2 For a Project Folder the Title field is mandatory. The Date Start will be automatically populated with the current date.
