C.3.4 Legacy data
Existing drawings must be stored and indexed (recording of all Title Block properties) in Vault using the appropriate import methods, eg Vault Autoloader.
The first time you open the legacy drawing, make sure that the file is checked out immediately. At check out the title block information is included in the Quick Start sheet.
 
  
Any missing (mandatory) information can be added either in Vault Explorer (see Vault - Edit Datasheet) or in AutoCAD:
The first time you save the existing data, the data sheet appears when not all in the Vault Client configuration defined properties exist or are filled. The datasheet displays in edit mode. See also Edit Datasheet / Title Block.
