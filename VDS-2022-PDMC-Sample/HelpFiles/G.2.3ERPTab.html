G.2.3 ERP Tab
We provide an ERP tab that shows you an example on how ERP related information can be visualised on a document in Vault, regardless of the category or document type.
In the directory. \ Templates \ Vault \ ERP you will find the guidelines on how to install the tab and use the ERP sample records.
The sample dataset (sample xml file) uses the same XML structure used when importing or exporting item data from Vault. The associated script file  reads from the XML elements. Change this file according to the structure of the available data from ERP. Direct querying to the ERP system is also possible (e.g. an SQL query or API access to the ERP System).
