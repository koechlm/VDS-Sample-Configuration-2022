V.4 Datasheet
All datasheets shows the properties of a document or a folder/project. All fields are read-only in this tab. To edit the properties you have to use the edit datasheet command in the context menu of the list view.
 
 
If a document is selected which has no matching category datesheet the standard properties list will be displayed.
