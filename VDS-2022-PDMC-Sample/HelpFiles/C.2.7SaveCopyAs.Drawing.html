C.2.7 Save Copy As | Drawing
Use Save Copy As to save Drawings to dwg, dxf or pdf.
The dialog follows the "Generate file numbers for drawing and presentation files" option:
 
Option Enabled:
1 - 7 The functionality of the dialog is the same as the new drawings dialog (option - enabled).
 
8 The Thumbnail of the drawing is added to the dialog.
 
Option Disabled:
1 - 7 The functionality of the dialog is the same as the new drawings dialog (option - disabled).
 
6 & 9 The File Name section automatic expands because the filename is populated and the number scheme is disabled.
 
note: the folder is set to the source file's location (1)
